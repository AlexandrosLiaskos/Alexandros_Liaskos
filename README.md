# Alexandros P. Liaskos - Interactive CV

This is an interactive, modular CV website for Alexandros P. Liaskos, built with HTML, Tailwind CSS, and JavaScript. It dynamically loads content components to create a smooth, single-page experience.

![image](https://github.com/user-attachments/assets/51f332f2-f50d-4e9f-960f-575325e0c40e)

## Features

- Responsive design using Tailwind CSS
- Modular content loaded dynamically via JavaScript
- Smooth animations powered by AOS (Animate On Scroll)
- Font Awesome icons and flag icons
- Ready for deployment on GitHub Pages

## Deployment

This site is designed to be hosted on GitHub Pages. It uses relative paths for all components and assets, and includes a `.nojekyll` file to ensure proper static file serving.

## Structure

- `index.html`: Main entry point
- `components/`: Contains modular HTML fragments for each CV section
- `js/loadComponents.js`: Script to dynamically load components
- `.nojekyll`: Disables Jekyll processing on GitHub Pages
